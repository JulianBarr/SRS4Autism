論語言發展時，需要包括語言理解和語言表達兩個次範籌。我們要聽得明白別人的說話，才能夠適切地回應，以達到傳情達意的效果，故此在討效、最直接的溝通方法。語言發展一般是指口語能力的發展，但由於在溝通的過程中，口語（又稱為說話或言語）是一種有聲符號，是利用語音去代表事物和思想，也是最有（如：說話），也可以是無聲的（如：文字、手勢、臉部表情等）。在不同的語言符號中，交流情感或進行思維活動。語言基本上是以符號代表事物和概念，符號可以是有聲的語言是人與人溝通的主要媒介，透過有系統地組織語言符號，我們可以相互交換訊息。

# 基本理論

學習目標，會與認知範疇內的目標相同。故此，語言能力和認知能力是不可分割的。基於上述原因，在語言範疇內有部分的字詞溝通，其中亦以名詞佔多數，其後漸漸增加動詞、形容詞等其它詞類。

力一般較理解能力遲三至四個月才出現，大概十三至十四個月大的兒童才能運用單字眼。初期理解的詞彙主要以名詞為主，繼而發展至動詞、形容詞等。至於表達能力，才能進入口語階段。

# 1. 語言前階段

兒童必須掌握這些基本的技能，及具備適當的語言理解來表達意思。因此，語言表達的初期訓練重點主要在建立兒童的發聲能力、動作或手勢溝通，以及模仿能力。在這階段，表達能力的發展是整體性的，兒童會運用聲音、語調、身體動作及手勢來表達意思。

視覺反應、社交反應、聽覺反應，以及建立語言前技能，如：配對技巧、遊戲技巧的視覺及聽覺接收是語言發展的基礎。故此，語言理解的初期訓練重點主要在建立語言資料（如：表情、手勢、動作、環境等）來對語言的整體行為作出理解。

兒童早期的語言發展約由嬰兒出生至十或十二個月期間進行，這階段的兒童依賴很多非語言的表達方式。

# 2. 單詞階段

當兒童能使用複雜的句子時，表示他們已掌握了詞彙運用與組句能力。所以當兒童能說出第一個有意義的單字詞時，表示他們已具備基本的概念和理解能力；在沒有概念基礎或理解基礎下，兒童不能說出有意義的字詞；在單字詞階段的兒童亦不能說出複雜的句子。

發展，故此，語言學習會以循序漸進的形式，按發展階段進行。由於語言的結構是按一定的規則組成，而語言的組織亦由簡至繁地發展，故此，語言的發展速度雖然會因個別情況而有差異，但發展的順序和模式卻大致是相似的。

# 1. 學習範疇-基本理論

兒童運用語言作描述的能力，例如：報告事件、說故事等。力後，訓練應著重發展兒童的抽象思維，例如：邏輯推理、解難能力等，以及發展習分析問題、定下假設、作出判斷及推論。因此，當兒童已建立良好的基礎語言能響。兒童會利用已掌握的詞彙和按語法規則，以具體的語言形式進行思維活動，學始跟思考力、想像力、記憶力、創造力等有著緊密的聯繫，彼此相輔相成，互相影響。

到了五至六歲，兒童已能掌握和運用許多語法規則。在這階段，兒童的語言能力開複合句與人溝通。一般來說，四歲的兒童已能明白部分常活的說話，及運用結構完整的簡單句和來，進而增加具修飾及語法功用的詞彙運用，例如：形容詞、數量詞、副詞等。將更多的詞彙連結成句，擴展語句的長度和複雜性，能將人、物、事的關係表達出來。

隨著詞彙的不斷增加，以及在短句運用對語法規則的熟習，三歲的兒童會進一步使用短句來表達需要。左右，兒童便會將詞彙組合成句，例如：動作詞和名詞、形容詞和名詞等，漸漸會在這階段，兒童能明白句子中不同詞彙之間的關係，以理解別的說話。到了兩歲，兒童的理解及表達能力會發展至句子階段。

語言發展期間，理解能力和表達能力是互相影響，並且互相配合的，兩者的發展程序和發展模式大致是相同的。

# 1. 粵語(口語)的應用

活動建議中的說話以書面語形式顯示，並在有需要時以經符號在括號內標示粵語的詞彙或句子。由於活動內容是以香港的社會環境及文化背景而設計，故訓練及活動設計均以口語化的粵語為藍本。

# 2. 詞彙的選取

作訓練時，宜先選取一些兒童常接觸的詞彙，再選擇一些較少接觸的詞彙。一般來說，兒童接觸得最多的事物，就是最早能掌握到的詞彙。

# 3. 教材的選擇

教材應以兒童熟悉及喜愛的物件為主，並要考慮兒童的能力以選取合適的教具。一般來說，兒童最容易掌握的為實物及玩具，進而是彩色相片或圖畫，黑白的線條圖則比較抽象，是較難掌握的教具。

# 4. 示範及提示的运用

在活動開始時，成人應給予清晰的示範。以語言表達的訓練為例，成人須示範說出目標詞彙或句子；而在語言理解的訓練中，成人在說出目標詞彙或句子後，須示範相關的行為（如：指向物件、進行句子中的意思等）。成人在示範後，可鼓勵兒童模仿，以確認兒童明白活動的做法及要求。進行活動時，成人需運用不同的提示方法，引導兒童學習。當兒童漸漸熟習後，人可逐步减少提示，並鼓勵兒童自行完成。
